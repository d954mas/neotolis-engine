## Decisions

- NT_GFX_ASSERT macro with __builtin_trap instead of assert() -- avoids Windows _wassert linker dependency
- Pool free preserves generation in slot id for correct realloc generation bump
- SLOT_IS_LIVE macro checks index bits match slot position to distinguish live from freed slots
- Stub backend create functions return 1 (nonzero) enabling headless test builds
- Default pool sizes: 32 shaders, 16 pipelines, 128 buffers
- Unified GL backend: single file handles WebGL 2 and OpenGL 3.3 Core via #ifdef NT_PLATFORM_WEB
- Desktop GL stub declarations: NULL function pointers satisfy compiler until glad loader is vendored
- Slot-based backend resource arrays for pipelines and buffers (parallel to shared pool)
- Pipeline stores vertex layout for re-applying attrib pointers on dynamic buffer bind
- NOLINT for GL-required int-to-ptr casts (glVertexAttribPointer, glDrawElements offset)
- Inline vector/matrix math instead of cglm -- avoids Windows clang UCRT DLL import issues with ASan
- __builtin_sqrtf/__builtin_fabsf for math -- maps to CPU instructions, bypasses CRT entirely
- Always-compiled test accessors (no #ifdef in .c) -- header guard controls visibility
- Consumer-provides-backend CMake pattern -- nt_shape links nt_core only, consumer provides nt_gfx or stub
- Billboard quad wireframe technique -- 4 verts/6 indices per edge, camera-facing, since glLineWidth > 1 is no-op in WebGL
- [Phase 16]: __builtin_sinf/__builtin_cosf builtins for tessellation math -- same UCRT bypass as sqrtf/fabsf
- [Phase 16]: Capsule tessellation: unified row-based UV with half_rings=segments/4 per hemisphere + tube section
- [Phase 16]: Internal emit helpers with optional quaternion pointer eliminate _rot code duplication
