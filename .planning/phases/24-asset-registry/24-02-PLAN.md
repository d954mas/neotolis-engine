  <action>
    Replace the stub implementations in nt_resource.c with full implementations:

    **nt_resource_mount():**
    1. Check initialized, return NT_ERR_INVALID_ARG if not
    2. Check pack_id not already mounted (scan packs[] for matching pack_id with mounted==1), return NT_ERR_INVALID_ARG if duplicate
    3. Find free slot in packs[] (mounted==0), return NT_ERR_INVALID_ARG if all 16 used
    4. Set pack_id, priority, pack_type=NT_PACK_FILE, mounted=1, asset_start=0, asset_count=0, blob=NULL, blob_size=0
    5. Set needs_resolve = true
    6. Return NT_OK

    **nt_resource_unmount():**
    1. Check initialized, return silently if not
    2. Find pack by pack_id (scan packs[] for matching pack_id with mounted==1), return silently if not found
    3. Mark all AssetMeta entries for this pack as unused: set resource_id = 0 for entries in range [pack.asset_start .. asset_start + asset_count)
    4. Clear pack slot: memset to zero (mounted becomes 0, blob becomes NULL)
    5. Set needs_resolve = true

    **nt_resource_set_priority():**
    1. Find pack by pack_id, return NT_ERR_INVALID_ARG if not found
    2. Update priority field
    3. Set needs_resolve = true
    4. Return NT_OK

    **nt_resource_request():**
    1. Check initialized, return NT_RESOURCE_INVALID if not
    2. Scan existing slots[1..] for matching resource_id -- if found, return existing handle (idempotent)
    3. If not found, allocate new slot from free_queue (pop from stack, bump generation, skip gen 0)
    4. Initialize slot: set resource_id, asset_type, gfx_handle=0, state=NT_ASSET_STATE_REGISTERED
    5. Set needs_resolve = true
    6. Return handle = resource_make(index, generation)

    **nt_resource_get():**
    1. If handle.id == 0, return 0
    2. Extract index = slot_index(handle), gen = generation(handle)
    3. If index == 0 or index > NT_RESOURCE_MAX_SLOTS, return 0
    4. If generations[index] != gen, return 0 (stale handle)
    5. Return slots[index].gfx_handle -- O(1) direct array read

    **nt_resource_is_ready():**
    1. Same validation as get()
    2. Return slots[index].state == NT_ASSET_STATE_READY

    **nt_resource_get_state():**
    1. Same validation as get()
    2. Return slots[index].state
    3. If invalid handle, return NT_ASSET_STATE_FAILED

    **nt_resource_step():**
    1. If !initialized or !needs_resolve, return immediately (O(1) fast path)
    2. For each active slot (slots[1..] where resource_id != 0):
       a. Find best AssetMeta entry: scan assets[] for matching resource_id, collect all with state == NT_ASSET_STATE_READY
       b. Among READY entries, pick the one whose pack has highest priority. Tiebreaker: highest pack index in packs[] array (last mounted gets higher index)
       c. If found: set slot.gfx_handle = best_meta.runtime_handle, slot.state = NT_ASSET_STATE_READY
       d. If no READY entry found: check if placeholder_handles[slot.asset_type] != 0, use placeholder. Set state to best available state from any matching entry (REGISTERED/LOADING/FAILED). If no entry at all, keep REGISTERED.
    3. Set needs_resolve = false

    **Important implementation notes:**
    - step() does NOT process a budget limit for resolve -- budget is for Phase 25 asset activation only. Resolve is metadata-only (no GPU work).
    - get() must be O(1): direct `slots[index].gfx_handle` read, no scanning.
    - request() scanning for existing slot is O(n) on slots but only called once per resource at init time, not per frame.
    - The "tiebreaker: last mounted wins" is implemented by iterating packs[] in order and the last match with equal priority wins (or by comparing pack array index).
